## Deployment from the beginning

1. 从gitee clone初始代码库
```bash
git clone https://gitee.com/hanxt/boot-security-starter.git
```
2. 安装boot-security-stater作为依赖项
```bash
# 注意不是commons-spring-boot-starter和basic-server，
# 在boot-security-stater目录下
mvn install 
```
